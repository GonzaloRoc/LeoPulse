API Explorer

Test onze live API-endpoints rechtstreeks vanuit uw browser.

GET `/iban/validate`
IBAN Validator

Voer een IBAN in om het formaat en de juistheid te valideren.

Response:
Klik op 'Test Endpoint' om het resultaat te zien.
GET `/kvk/validate`
KvK Validator

Voer een Nederlands KvK-nummer (8 cijfers) in om het formaat te valideren.

Response:
Klik op 'Test Endpoint' om het resultaat te zien.
POST `/anomaly/detect`
Anomalie Detector

Voer een komma-gescheiden lijst met getallen in om uitschieters te vinden.

Response:
Klik op 'Test Endpoint' om het resultaat te zien.

Klik op 'Test Endpoint' om het resultaat te zien.
