Authenticatie

SnelAPI gebruikt API Keys om requests te authenticeren. U kunt uw API-sleutels bekijken en beheren in de SnelAPI Portal.

Voeg uw API-sleutel als volgt toe aan de request header:

Authorization: Bearer UW_API_SLEUTEL

Endpoints

Dit gedeelte beschrijft de beschikbare API-endpoints, inclusief de vereiste parameters en voorbeeldantwoorden.

GET /resource

Haalt een lijst met resources op.

curl -X GET "https://api.snelapi.nl/v1/resource" \
-H "Authorization: Bearer UW_API_SLEUTEL"

Foutcodes

De API gebruikt conventionele HTTP-statuscodes om het succes of falen van een request aan te geven.

  • 200 OK - Het request was succesvol.
  • 401 Unauthorized - Uw API-sleutel is onjuist of ontbreekt.
  • 404 Not Found - De opgevraagde resource bestaat niet.
